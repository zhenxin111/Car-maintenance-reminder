## 作业使用说明

### 文件内容

本作业包含三个文件：
+ index.html 作业程序入口
+ carReminder.js javascript程序实现
+ Readme.txt 作业说明

### 使用步骤：

+ 作业上传形式为压缩包，解压至文件夹；
+ 作业程序入口文件为index.html，可右击使用Chrome浏览器打开（也可以是其他主流浏览器）；
+ 本作业使用网页的表单输入方式，进行交互测试；
+ 只需将测试用例复制粘贴至表单内，点击提交按钮，即可得到程序输出结果；
+ 重复测试前，点击"清空输出"按钮，清空上一次的输出内容；

### 作业反馈：

在文档中，没有说明报废提醒的准确时间范围（是否包括已开始报废的车辆），但是经由测试用例表明，我认为本次作业所指报废提醒周期是：报废时间的前一个月（不考虑日）—————— 报废时间（不含）；我的作业程序也是按照上面的标准所编写，如有误差，希望收到您的反馈，谢谢！
